Provided by the octree module.

Defined in src\core\raycasting.js:38

Finds the entry plane of the first octant that a ray travels through.

Determining the first octant requires knowing which of the t0's is the largest. The tm's of the other axes must also be compared against that largest t0.

Parameters:

  • tx0 Number
    • Ray projection parameter.
  • ty0 Number
    • Ray projection parameter.
  • tz0 Number
    • Ray projection parameter.
  • txm Number
    • Ray projection parameter mean.
  • tym Number
    • Ray projection parameter mean.
  • tzm Number
    • Ray projection parameter mean.

Returns:

Number:

The index of the first octant that the ray travels through.

Finds all octants that intersect with the given ray.

Parameters:

  • octant Octant
    • The current octant.
  • tx0 Number
    • Ray projection parameter. Initial tx0 = (minX - rayOriginX) / rayDirectionX.
  • ty0 Number
    • Ray projection parameter. Initial ty0 = (minY - rayOriginY) / rayDirectionY.
  • tz0 Number
    • Ray projection parameter. Initial tz0 = (minZ - rayOriginZ) / rayDirectionZ.
  • tx1 Number
    • Ray projection parameter. Initial tx1 = (maxX - rayOriginX) / rayDirectionX.
  • ty1 Number
    • Ray projection parameter. Initial ty1 = (maxY - rayOriginY) / rayDirectionY.
  • tz1 Number
    • Ray projection parameter. Initial tz1 = (maxZ - rayOriginZ) / rayDirectionZ.
  • raycaster Raycaster
    • The raycaster.
  • intersects Array
    • An array to be filled with the intersecting octants.
